# Get Full Match <Badge type="info" text="GET"/>

## `getFullMatch`

Returns the full match.

::: tip INFO
This is not an official endpoint on the osu!api.
:::

## Usage

`api.getFullMatch(matchId, token)`

### Parameters

`matchId` <small>String</small><br>
The ID of the match.

`token` <small>String</small><br>
Access Token
